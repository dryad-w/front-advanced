## InputElement
- WhiteSpace 空白
  - \<TAB>
  - \<VT>
  - \<FF>
  - \<SP>
  - \<NBSP>
  - \<ZWNBSP>
  - \<USP>
- LineTerminator 换行符
  - \<LF> '\n'
  - \<CR> '\r'
  - \<LS>
  - \<PS>
- Comment 注释
  - MultiLineComment
  - SingleLineComment
- Token
  - Punctuator (= < ...)
  - IdentifierName
    - Keywords (for let ...)
    - Identifier (document ...)
      - 变量名 （不能跟Keywords重合）
      - 属性名
    - Future reserved Keywords: enum
  - Literal (1 '2' true ...)
    - Number
    - String
    - Boolean
    - Object
    - Null
    - Undefined
    - Symbol
